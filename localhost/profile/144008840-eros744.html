Posted on 09-09-18, 01:49 pm in impossible Importing a NSMBU Mario model
It's more of a random... meme thing or something like that.
It makes marios eyes look like derpy (like the SMG4 videos for example)
and replaces some of the NSMBU tiles with the SMB ones as well as the SMB ones with the NSMBU ones.
Installation instructions:
FIRST: Be sure to have Luma's game patching enabled! Just extract the zip file at root of the sd card (works only in luma with this method),and if you have another region instead of EUR, you will have to change the folder name inside /luma/titles/[this one] to the title id of your region
If you want to disable a mod
Just delete that mod file inside of /luma/titles/[yourregiontitleid]/romfs/
The derpy mario mod is the WU_Player_MarioMdl.szs file inside the /Models/ folder.
The SMB1 replacement is the M1_field_plain_textures.ctpk file inside the /TileData/ folder.
The NSMBU replacement is the WU_field_plain_textures.ctpk file inside the /TileData/ folder.
To enable Luma game patching (if you didn't already)
Be sure to have Luma installed in your 3DS (obviously)! Hold Select when turning on the 3DS (any model, New 3DS, XL models, 2DS) until the Luma3DS configuration menu appears, the press the down button until you are on the ( ) Enable game patching option, and press A to turn it on, and X will appear, now just press START and the 3DS/2DS/NEW 3DS/NEW 2DS will reboot, just wait normally until the 3DS home screen appears and open Super Mario Maker (again, looks like I'm captain obvious), Now it should work!
